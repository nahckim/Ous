# Ous Commands

| Command | Action | Routes To |
|---|---|---|
| onote "text" | Raw capture | ledger direct |
| oguide "text" | Structured note | Ollama |
| oask "prompt" | Query | DeepSeek API |
| odream | Trigger dream | dreaming module |
| ous | Start/restart | cargo run |

## Aliases (deprecated)
cnote, cguide, cask, cdream still work, prefer o-prefix
